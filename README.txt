== Description ==

It's a simple plugin that calculates the remaining reading time as you scroll down. In order for it to work, all it takes is activating it, and the plugin will run without further ado. Also, this plugin operates over the post, rather than over the website.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp-reading-time.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done!
